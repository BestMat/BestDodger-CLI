Windows Vista Virtual Machine Documentation

  1. Download Python 3, Java JDK 11.0.7+, NodeJS 12+
  2. Click to download VM package
  3. Open terminal/shell/command prompt and type the following commands
  4. pip install pywebview
  5. pip install tkinter
  6. Extract the zip file downloaded
  7. Open chrome/firefox/ie folders to choose the vm's browser
  8. Run the main.py file (python main.py)
